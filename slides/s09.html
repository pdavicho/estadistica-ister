¿Qué es una Medida de Tendencia Central? 2 / 18
📖 Definición
Una medida de tendencia central es un valor que representa al conjunto de datos: el punto alrededor del cual se agrupan. Resume toda una distribución en un solo número "típico".
  • En el Tema 2 vimos la forma de los datos (tablas, histogramas). Ahora buscamos su centro.
  • Responde preguntas como: ¿cuánto gana "normalmente" un empleado? ¿cuál es la nota "típica"?
  • Las tres clásicas: media, mediana y moda. Cada una mide el centro de forma distinta.
💡 Idea clave de la semana
No siempre coinciden. Elegir la medida correcta depende de la forma de los datos y de si hay valores atípicos.

9.1
La Media Aritmética (x̄)
El promedio: la medida más conocida y la base de casi toda la estadística posterior.
Media Aritmética: Definición y Fórmula 5 / 18
📖 Definición
La media es el resultado de sumar todos los valores y dividir entre el número de datos. Es el "punto de equilibrio" de la distribución.
x̄ = (x₁ + x₂ + … + xₙ) / n = Σxᵢ / n
Σ (sigma) significa "suma de todos"; n es el tamaño de la muestra

✅ Ventajas

Usa toda la información; es la base de la varianza, la desviación y casi todo lo que sigue.

⚠️ Desventaja

Es muy sensible a valores atípicos: un solo dato extremo la desplaza mucho.

Ejemplo: Sueldos de un Equipo (11 personas) 6 / 18

📋 Sueldos mensuales (USD), ordenados — los usaremos toda la clase

780 · 850 · 850 · 900 · 950 · 1000 · 1000 · 1000 · 1100 · 1200 · 4500

n = 11 · el último (4500) es el sueldo del gerente

x̄ = 14130 / 11 = 1284.5 USD
Suma = 780+850+850+900+950+1000+1000+1000+1100+1200+4500 = 14130
⚠️ Mira el resultado
La media dice 1284.5 USD, pero 9 de los 11 empleados ganan 1100 o menos. El sueldo del gerente (4500) "infla" la media y la vuelve poco representativa del equipo.

9.2
La Mediana (Me)
El valor del medio: divide los datos ordenados en dos mitades iguales y resiste a los atípicos.
Mediana: el Valor Central 8 / 18
📖 Definición
La mediana es el valor que ocupa la posición central cuando los datos están ordenados. Deja el 50% de los datos por debajo y el 50% por encima.

🔢 Si n es impar

La mediana es el dato de la posición (n+1)/2. Es un valor real del conjunto.

🔢 Si n es par

Es el promedio de los dos centrales: posiciones n/2 y n/2 + 1.

💡 Paso imprescindible
Siempre ordena los datos de menor a mayor antes de buscar la mediana. Sin ordenar, el valor central no significa nada.

Mediana del Ejemplo de Sueldos 9 / 18

Contamos hasta el 6º dato de la lista ordenada:
780850850900950100010001000110012004500
Me = 1000 USD
El 6º valor; 5 sueldos quedan por debajo y 5 por encima
💡 Compara con la media
Mediana = 1000 vs media = 1284.5. La mediana ignora el sueldo extremo del gerente y describe mucho mejor al empleado "típico". Por eso para sueldos se suele reportar la mediana.
⚠️ Caso n par (mini-ejemplo)
Para 3, 5, 7, 8 (n = 4): los dos centrales son 5 y 7 → Me = (5+7)/2 = 6.

9.3
La Moda (Mo)
El valor más frecuente: la única medida que también sirve para datos cualitativos.
Moda: el Valor Más Frecuente 11 / 18
📖 Definición
La moda es el valor (o categoría) que aparece con mayor frecuencia. Es la única medida que se puede usar con datos cualitativos (ej. el color más vendido).
780850850900950100010001000110012004500
Mo = 1000 USD
El valor 1000 aparece 3 veces; ningún otro se repite tanto

1️⃣ Unimodal

Una sola moda (nuestro caso).

2️⃣ Bimodal

Dos valores empatados en frecuencia.

0️⃣ Amodal

Ningún valor se repite → sin moda.

9.4
¿Cuál Medida Usar?
Media, mediana y moda no compiten: cada una responde mejor según la forma de los datos.
Comparación y Efecto de los Atípicos 13 / 18
MedidaÚsala cuando…AtípicosNuestro ej.
Media x̄Los datos son simétricos, sin extremosMuy sensible1284.5
Mediana MeHay sesgo o valores atípicos (sueldos, precios)Resistente1000
Moda MoDatos cualitativos o el valor más comúnResistente1000
⚠️ La media "se va" con el atípico
El sueldo del gerente (4500) arrastró la media a 1284.5, pero mediana y moda se quedaron en 1000, fieles al equipo. Regla práctica: si media y mediana difieren mucho, hay sesgo o atípicos → reporta la mediana.
💡 Relación con la forma (Semana 8)
Simétrica → media ≈ mediana ≈ moda. Sesgo a la derecha → media > mediana > moda. Sesgo a la izquierda → al revés.

9.5
Cálculo con Python y R
Las tres medidas se obtienen con funciones directas en ambos lenguajes.
🐍 Python: media, mediana y moda 15 / 18
import pandas as pd
import statistics as st

sueldos = [780,850,850,900,950,1000,1000,1000,1100,1200,4500]
s = pd.Series(sueldos)

print("Media  :", round(s.mean(), 2))   # promedio
print("Mediana:", s.median())            # valor central
print("Moda   :", s.mode()[0])           # más frecuente

# Sin pandas, con el módulo statistics:
print(st.mean(sueldos), st.median(sueldos), st.mode(sueldos))

📊 R: mean(), median() y la moda 16 / 18
sueldos <- c(780,850,850,900,950,1000,1000,1000,1100,1200,4500)

mean(sueldos)     # 1284.545  -> media
median(sueldos)   # 1000      -> mediana

# R no trae una función de moda: la definimos
moda <- function(x){
  ux <- unique(x)
  ux[which.max(tabulate(match(x, ux)))]
}
moda(sueldos)     # 1000

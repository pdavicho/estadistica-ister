🎲 2.2 Muestreo Probabilístico

📖 Definición
Cada elemento tiene una probabilidad conocida y mayor que cero de ser seleccionado. Garantiza representatividad y permite calcular el error muestral.
Probabilístico

🎯 Aleatorio Simple (MAS)

Cada elemento tiene la misma probabilidad. Se usa un generador de números aleatorios.

Ej: Seleccionar 200 clientes de 10,000 con random.sample().

P = n/N
Probabilístico

📏 Sistemático

Inicio aleatorio, luego cada k-ésimo elemento de la lista ordenada.

Ej: En 1,000 servidores, seleccionar cada 10mo (k=10).

k = N/n
Probabilístico

📊 Estratificado

División en estratos homogéneos. Se muestrea proporcionalmente de cada uno.

Ej: Encuestar por depto (TI, Ventas, RRHH) proporcionalmente.

nₕ = (Nₕ/N)×n
Probabilístico

🗂 Por Conglomerados

Grupos heterogéneos (conglomerados). Se seleccionan grupos completos al azar.

Ej: Seleccionar 5 sucursales bancarias al azar y estudiar a todos sus empleados.

Grupos = m

🔍 Ejercicio de Identificación

"Una empresa divide sus 50,000 clientes en 4 regiones y toma muestras proporcionales de cada región."

⚠️ 2.3 Muestreo No Probabilístico

⚠️ Atención
No todos los elementos tienen la misma probabilidad. Más rápidos y económicos, pero introducen sesgo potencial. Se usan en estudios exploratorios.
No Probabilístico

⚡ Por Conveniencia

Elementos más fácilmente accesibles. Rápido pero con alto riesgo de sesgo.

Ej: Encuestar solo a usuarios que visitan el sitio el lunes por la mañana.

No Probabilístico

📋 Por Cuotas

Cuotas por características (edad, género), pero la elección dentro de cada cuota es discrecional.

Ej: Entrevistar 50 hombres y 50 mujeres sin importar cómo se seleccionan.

No Probabilístico

❄️ Bola de Nieve

Participantes reclutan a nuevos. Útil para poblaciones de difícil acceso.

Ej: Estudiar freelancers: cada uno recomienda a otro colega.

🔄 ¿Cuándo usar cada tipo?

  • Prob.: Inferencia estadística formal
  • No prob.: Estudios piloto, exploración inicial
  • Siempre: Justifica tu elección

📐 2.4 Tipos de Variables

🏷 Cualitativas (Categóricas)

Expresan cualidades o atributos. No se operan matemáticamente.

Nominales: Sin orden — Ej: SO (Windows/Linux), color
Ordinales: Con orden — Ej: Prioridad ticket (Crítico/Alto/Bajo)

🔢 Cuantitativas (Numéricas)

Expresan cantidades. Permiten operaciones matemáticas.

Discretas: Enteros contables — Ej: N° errores en código, N° clientes
Continuas: Cualquier valor — Ej: Tiempo respuesta (ms), temperatura CPU

🔍 Ejercicio

"Número de tickets de soporte resueltos por un técnico en una semana."

📏 2.5 Escalas de Medición (NOIR)

EscalaCaracterísticasOperacionesEjemplo en Sistemas
🏷 NominalSolo categorías sin ordenModa, frecuenciasTipo SO: Windows, Linux, macOS
📶 OrdinalCategorías con orden. Distancia desconocida.Moda, MedianaPrioridad: Crítico > Alto > Bajo
🌡 IntervaloOrden + distancias iguales. Cero arbitrario.Media, Desv. std.Temperatura servidor (°C)
⚖️ RazónOrden + distancias + Cero absoluto.Todas. Cocientes.Tiempo respuesta (ms). 0ms = sin latencia
💡 Regla práctica
Nominal → Ordinal → Intervalo → Razón (de menor a mayor precisión). Una escala superior puede usarse como inferior, pero no viceversa.

🐍 2.6 Python: Técnicas de Muestreo

import numpy as np
np.random.seed(42)
N, n = 1000, 50

# 1. ALEATORIO SIMPLE
mas = np.random.choice(range(1, N+1), size=n, replace=False)
print(f"MAS (primeros 5): {sorted(mas)[:5]}")

# 2. SISTEMÁTICO
k = N // n  # k = 20
inicio = np.random.randint(1, k+1)
sistematico = [inicio + k*i for i in range(n)]
print(f"Sistemático k={k}, inicio={inicio}: {sistematico[:5]}")

# Comparación de medias
salarios = np.random.normal(800, 150, N)
print(f"\nMedia real:        ${salarios.mean():.2f}")
print(f"Media MAS:         ${salarios[mas-1].mean():.2f}")
print(f"Media Sistemático: ${salarios[[i-1 for i in sistematico]].mean():.2f}")
import pandas as pd
import numpy as np
np.random.seed(42)

deptos = {'TI':300,'Ventas':250,'RRHH':150,'Finanzas':200,'Marketing':100}
filas = []
for d, c in deptos.items():
    for sal in np.random.normal(800, 150, c):
        filas.append({'departamento': d, 'salario': round(sal, 2)})
df = pd.DataFrame(filas)

# Muestreo estratificado proporcional (10% de cada estrato)
n_total, N = 100, len(df)
muestra = pd.DataFrame()
for d, c in deptos.items():
    n_estrato = round(c/N * n_total)
    muestra = pd.concat([muestra, df[df.departamento==d].sample(n_estrato)])
    print(f"  {d}: {n_estrato} empleados ({c/N*100:.0f}%)")

print(f"\nMedia real:    ${df.salario.mean():.2f}")
print(f"Media muestra: ${muestra.salario.mean():.2f}")
Output
MAS (primeros 5): [10, 45, 98, 133, 175]
Media real: $799.87 | Media MAS: $803.21 | Media Estrat.: $800.12

📊 2.7 R: Técnicas de Muestreo

set.seed(42)
N <- 1000; n <- 50
salarios <- rnorm(N, mean = 800, sd = 150)

# 1. Aleatorio Simple
mas <- sample(1:N, n, replace = FALSE)

# 2. Sistemático
k <- N %/% n  # k = 20
inicio <- sample(1:k, 1)
sistematico <- seq(inicio, N, by = k)

cat("=== COMPARACIÓN ===\n")
cat(sprintf("Media real:        $%.2f\n", mean(salarios)))
cat(sprintf("Media MAS:         $%.2f\n", mean(salarios[mas])))
cat(sprintf("Media Sistemático: $%.2f\n", mean(salarios[sistematico])))
library(dplyr)
set.seed(42)

deptos <- rep(c("TI","Ventas","RRHH","Finanzas","Marketing"), c(300,250,150,200,100))
df <- data.frame(departamento = deptos, salario = rnorm(1000, 800, 150))

# Muestreo estratificado proporcional (10% por estrato)
muestra <- df |>
  group_by(departamento) |>
  slice_sample(prop = 0.1) |>
  ungroup()

cat(sprintf("Población: %d | Muestra: %d\n", nrow(df), nrow(muestra)))
cat(sprintf("Media real:    $%.2f\n", mean(df$salario)))
cat(sprintf("Media muestra: $%.2f\n", mean(muestra$salario)))

# Distribución por estrato
muestra |> count(departamento) |> print()
Output
Media real: $799.87 | Media MAS: $803.21 | Media Estratificado: $800.12

🎯 2.8 Puntos Clave y Quiz

Puntos Clave Semana 2

1
El muestreo probabilístico garantiza representatividad y permite inferencia; el no probabilístico es más rápido pero introduce sesgo.
2
El estratificado es el más preciso cuando la población tiene subgrupos distinguibles.
3
Las variables cualitativas describen atributos; las cuantitativas miden cantidades. Solo las cuantitativas permiten operaciones aritméticas.
4
Las escalas NOIR determinan qué métodos estadísticos aplicar. La escala de Razón es la más completa.
5
Python: DataFrame.groupby().sample() · R: dplyr::group_by() |> slice_sample()

Caso: DataSoft Ecuador — Encuesta de Satisfacción

ABP · 25 min
Escenario: DataSoft Ecuador tiene 2,000 empleados en 4 departamentos: Desarrollo (800), QA (400), Operaciones (500), Administración (300). Se quiere encuestar 200 empleados para medir satisfacción laboral con el nuevo sistema de gestión.
  1. 1
    ¿Cuál es la población N y la muestra n?
    N = 2,000 empleados. n = 200 empleados a encuestar.
  2. 2
    Calcula cuántos empleados de cada departamento en muestreo estratificado proporcional.
    Desarrollo: 800/2000×200 = 80 · QA: 400/2000×200 = 40 · Operaciones: 500/2000×200 = 50 · Admin: 300/2000×200 = 30
  3. 3
    Si aplicas muestreo sistemático, ¿cuál es el valor de k?
    k = N/n = 2000/200 = 10. Se selecciona cada 10mo empleado de la lista.
  4. 4
    El jefe propone "encuestar solo a quienes estén disponibles ese día." ¿Qué tipo de muestreo y qué sesgo introduce?
    Muestreo por Conveniencia (No Probabilístico). Sesgo: quienes están disponibles pueden ser empleados con poco trabajo, excluyendo a los más ocupados que pueden tener menor satisfacción.
  5. 5
    "Satisfacción: Muy insatisfecho / Insatisfecho / Neutral / Satisfecho / Muy Satisfecho." ¿Qué escala de medición y tipo de variable es?
    Escala Ordinal (hay orden pero no sabemos si la distancia entre categorías es igual) · Variable Cualitativa Ordinal.
